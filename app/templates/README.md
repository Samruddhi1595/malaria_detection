index.html

This file provides the frontend user interface for the Malaria Detection System. It allows users to upload microscopic blood cell images and view malaria detection results generated by the deep learning model.

The interface is built using HTML, CSS, and JavaScript and communicates with the Flask backend.

Interface Components

Navigation Bar
Displays the system name MalariaScope Neural Diagnostic Platform and indicates the system status.

Hero Section
Introduces the AI-powered malaria detection system with information about:

Model accuracy

Processing speed

Number of scans processed

Image Upload System
Users can upload blood cell images through:

Drag-and-drop interface

File selection dialog

Supported image formats include:

PNG

JPG

JPEG

File Preview
Before analysis, the system displays:

Image preview

File name

File size

Image Analysis
Once the image is uploaded:

The Analyze Sample button sends the image to the backend /predict API using a POST request.

The system displays a loading spinner while processing.

Prediction Result Page
After analysis, the interface displays a diagnostic report including:

Prediction result (Parasitized / Uninfected)

Confidence score

Probability values for each class

Processing time

Timestamp of the analysis

Visual indicators are used:

Green color for Uninfected

Red color for Parasitized

Medical Disclaimer
The interface includes a disclaimer indicating that the system is intended as a screening tool and should not replace professional medical diagnosis.

New Scan Option
Users can reset the interface and upload another image for analysis.
