# malaria_detection
Malaria Detection Using Deep Learning

This project focuses on detecting malaria-infected cells from microscopic blood smear images using deep learning techniques. The dataset contains images of parasitized and uninfected red blood cells. Data visualization was performed to understand the dataset distribution and sample cell structures. A convolutional neural network (CNN) model is used to learn patterns in the images and classify them as infected or healthy.

The goal of this project is to demonstrate how computer vision and deep learning can assist in the early detection of malaria and support medical diagnosis.
